Gestion biens immobiliers – Laravel & PHP

Description

Application web complète pour gérer les biens à vendre et à louer. Recherche avancée avec filtres, affichage détaillé, gestion des contacts. Interface admin intuitive pour gérer les biens, les options et les locations.

Technologies utilisées

  • PHP & Laravel
  • Blade (templating)
  • MySQL
  • Bootstrap 5
