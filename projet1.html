Gestion de commandes – Java Spring Boot

Description

Application complète avec interface administrateur et utilisateur : gestion des produits, panier, suivi des commandes en temps réel. Authentification sécurisée avec Spring Security (ADMIN/USER), pages de login et inscription.

Technologies utilisées

  • Java 17
  • Spring Boot & Spring Security
  • Thymeleaf
  • MySQL
  • Bootstrap 5
